# Publish Templates

## Local Markdown Template

```markdown
# {Title}

## Executive Summary
{One paragraph}

## Background
{Context and goals}

## Change Logic
{How and why the change works}

## Code Impact
- {file/path/a}: {impact}
- {file/path/b}: {impact}

## Risks and Mitigations
- Risk: {risk}
  - Mitigation: {mitigation}

## Validation Plan
- {test or check 1}
- {test or check 2}

## Diagram ({mermaid|draw})
{diagram content}

## Destination Metadata
- Destination: local
- Output Path: {absolute path}
```

## Yuque Markdown Payload Template

```markdown
# {Title}

> Source: generated by p9-dev-doc-summarizer
> Updated At: {ISO-8601 datetime}

## Executive Summary
{One paragraph}

## Background
{Context and goals}

## Change Logic
{How and why the change works}

## Code Impact
- {module/file}: {impact}

## Risks and Mitigations
- {risk and mitigation}

## Validation Plan
- {checks}

## Diagram
{diagram content}
```

Required parameters before Yuque publish:
- repo/space identifier
- document title
- create vs update
- credential method (token/cli/session)

If publish is blocked, output this payload to local Markdown and provide the exact command the user should run next.
